## 4. インタラクティブ要素・アニメーション仕様

### グローバルアニメーション

**エントリアニメーション:**
- fadeInUp: 下から上へ（0.8s）
- slideInLeft: 左から右へ（0.8s）

**スクロール進入アニメーション:**
- 各スライドはスクロール進入時に自動アニメーション

**ホバーエフェクト:**
- カード: translateY(-8px)、box-shadow強化
- テキスト: 色変化、下線アニメーション

### グラフアニメーション仕様

- **棒グラフ**: 下から上へ成長（800ms）
- **折れ線グラフ**: 左から右へ描画（1000ms）
- **ドーナツ**: 360度回転（1200ms）
- **数字カウント**: 0からターゲット値へカウントアップ（800ms）

## 5. 実装方針

### フレームワーク＆ライブラリ

| 用途 | 推奨ライブラリ |
|---|---|
| フレームワーク | React (Next.js) |
| グラフ | Recharts / Chart.js |
| スクロールアニメーション | AOS / Intersection Observer API |
| UI コンポーネント | shadcn/ui / Headless UI |
| アイコン | Lucide React / Font Awesome |
| Canvas描画 | Canvas API / p5.js |
