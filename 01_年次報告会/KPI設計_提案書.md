## 2. データテーブルに基づく現状分析 (GAP Analysis)

同条件（2026.1-3期間・60分換算）でKANEBOを加味して比較した結果、マツキヨココカラライブの立ち位置と競合との乖離（ギャップ）がより立体的になりました。

*   **【強み】視聴者数（リーチ）:**
    ロート製薬（871人）やKANEBO（1,458人）、アットコスメの企画（2,494人）と比較しても、5,000人超えという**資生堂を除き、一般メーカー比較で最大クラスのリーチ力**を持っています。事前の集客導線は非常にうまくいっています。
*   **【課題】平均視聴分数（リテンション）:**
    マツキヨココカラライブは直近で **わずか 3.09分** であり、競合で最も低い水準です。ロート製薬や資生堂（約10〜12分）と比較しても1/3未満であり、KANEBO（28.2分）のような「視聴者を釘付けにする構成」ができておらず、深刻な早期離脱が起きています。
*   **【課題】商品CTR（アクション）:**
    マツキヨココカラライブは **1.04%**。一般メーカーであるロート製薬（5.3%）や、高いエンゲージメントを誇るKANEBO（11.7%）を大きく下回っており、商品への「クリック誘導」が機能していません。
*   **【課題】コメント率（エンゲージメント）:**
    マツキヨココカラライブは **0.71%** と最低水準です。KANEBO（9.1%）などのように「参加感」を高める工夫が不足していることが、早期離脱を引き起こしている最大の要因と考えられます。

### 参考：各ブランドの配信の強みと特徴（定性要因）

データテーブルの背景にある、各社の配信戦略上の工夫と強みです。

*   **マツキヨココカラライブ (内部全体):**
    *   **【強み】圧倒的な集客力（リーチ）:** プッシュ通知・メルマガ・公式アプリなどマツキヨ本体の顧客基盤を活用し、毎回5,000人超の視聴者を集めている（一般メーカー比較では最大クラス）。事前集客導線は非常に機能している。
    *   **【課題】配信構成が「商品紹介中心・受け身型」で離脱を招く:** 配信冒頭から商品説明モードに入りやすく、視聴者を引き留める「双方向の仕掛け」（参加型コンテンツ・問いかけ・休憩コーナー等）が弱い。結果として最初の3分で大量離脱が起きている（A2分析）。
    *   **【課題】お悩み訴求が59%の配信で弱い（自分ごと化の不足）:** 直近44配信の台本分析（#48〜#92）では、お悩みワードが3つ以上明示されている配信は36%にとどまり、59%が「商品名・ブランド名のみ訴求」「機能訴求はあるが悩みが不在」の状態。視聴者が「自分の悩みだ」と感じる仕掛けが不足している（A6分析）。下期データでは、お悩み訴求が強い配信のCTRが+10.8%上回ることが確認されている。
    *   **【課題】CTR誘導の仕組みが弱い（クーポン・限定性の不足）:** 「今すぐ買う理由」（限定クーポン・タイムセール・限定数量など）を配信内で提示する型化が未実装。視聴者は商品を知っても「後で買えばいい」と離脱する構造（A2分析）。2月配信ではクーポン活用でCTRが上昇する事例があり、再現性のある型化が課題。
    *   **【課題】出演者個人のSNSフォロワー基盤が小さい:** 資生堂PBP（個人IG最大16万人・累計99万人）やアットコスメ公式TikTok（13.7万人）と比較し、出演者個人発信での集客チャネルが弱い。LIPS連動回（外部インフルエンサー起用）では視聴者数が跳ねる傾向があり、外部連動・出演者育成の余地が大きい（A2・A5分析）。
    *   **【参考】Xフォロリポ施策の効果限定:** リピーター数・チャット数は大幅増（+60〜81%）の一方、平均視聴分数は▼43%、商品クリック数は▼15〜31%と購買誘導には逆効果。「ファン拡散・X上での拡散」と「購買転換」を別KPIで設計する必要がある（A5分析）。
*   **資生堂:**
    *   資生堂は競合比較において全体的に数値が突出して高い傾向にあります。
    *   **リーチの最大化:** Instagramのサイマル配信を活用してFireworkプレーヤーへの流入を促進しており、Instagramから自社ホームページへと視聴習慣を定着させている点が大きな要因と考えられます。
    *   **属人的なエンゲージメント:** Instagramのフォロワーが多い美容部員が出演することで、美容部員個人のファンがライブに流入し、コメント数や視聴時間を押し上げています。また3月18日からは美容部員個人発信のライブも開始しています。
    *   **販促連動 (CTR):** クーポンの提供、新商品の打ち出し、「ここでしか買えない」といった限定性の訴求を組み合わせることで、視聴者の購買意欲を高めています。
*   **KANEBO (花王):**
    *   **高リテンション・高エンゲージメント:** 視聴時間とCTR・コメントといったエンゲージメント指標が特に高いのが特徴です。
    *   **マルチチャネルでの集客:** LINE通知・IGサイマル配信・スプロケット（配信中にライブ映像を表示）という複数のチャネルを組み合わせて視聴者を集めています。
    *   **ブランド主導のコミュニティ:** 美容部員個人のInstagramアカウントに依存しているわけではなく、KANEBOというブランド自体が熱狂的なブランド愛好家から支持されており、ライブを通じてそのファンが定着していることが背景にあります。
    *   **純粋なコンテンツ力に基づくCTR:** クーポンはほぼ使用しておらず、商品やコンテンツそのものの魅力で購買行動を引き出している点が資生堂との大きな違いと言えます。
*   **ロート製薬:**
    *   **月1回の定期配信「ロートハートライブ」:** 月1回ペースで定期開催。配信数は3ヶ月で3回と少ないが、リピーター比率16.5%・リピーターエンゲ率61.9%を実現しており、**頻度よりも「定期性」がリピーター育成に効いている**可能性がある。
    *   **複数ブランドの横断紹介:** 糀肌・エピステーム・オバジなどロート製薬グループの複数ブランドを1配信で横断紹介。ブランド軸ではなく「ロート製薬」という企業ブランドでファンを集める設計。
    *   **ライブ限定キット商品で視聴インセンティブを提供:** 「ライブでしか買えない限定キット」を販売することで、視聴の動機付けと購買誘導を同時に実現。リピーターCTR5.3%は競合4社の中では最も低いものの、マツキヨココカラライブ（**リピーター層のCTR** 3.87%）よりは高い水準。
    *   **初回配信の予想超えの実績:** 初回配信で「予想10人に対して400人超の視聴者」を集めた事例があり（ロート製薬オンライン公式）、限定性とコミュニティ告知の組み合わせで初動を作れている。
    *   **【要確認】定性情報は限定的:** 公開情報が少なく、出演者・運営体制・告知チャネルの詳細は要確認。Firework経由でのヒアリング機会があれば優先的に深掘りすべき。
*   **アットコスメ:**
    *   **「自分ごと化」を促す構成設計:** 単純な商品紹介にとどまらず、「セルフチェック機能」「原因解説」を配信中に挿入し、視聴者が自分の悩みと商品を結びつけるプロセスを強化している点が高エンゲージメント（リピーターコメント率42.3%・リピーターCTR49.4%は5社中1位）の最大要因。
    *   **「中の人」コメント参加とコミュニティ醸成:** 運営スタッフも「中の人」として常連視聴者と双方向のやりとりを行い、配信中の一体感を形成。配信合間に意図的な「休憩コーナー」を設けることでリピーター層の楽しみ要素として機能している（出典：スタジオ見学・情報交換会ヒアリング）。
    *   **メーカー担当者のコメント参加:** 体験会・ライブ配信中にメーカー担当者が直接コメント欄に参加し、視聴者の質問にリアルタイムで回答。コメント数増加と納得感による購買促進を両立している。
    *   **毎週金曜20時×4年継続の視聴習慣形成:** 2021年12月開始、毎週金曜20時の固定スケジュールで4年継続。年間累計100万人超の視聴実績を持ち、リピーター比率50.5%は5社中1位（マツキヨココカラライブの10.1倍）（出典：@cosme公式料金メディアガイド 202604_202606_atcosmeBA_promotion.pdf）。
    *   **100%内製・プロ機材による制作体制:** OBS配信＋業務用カメラの3カメスイッチング＋簡易テロップを活用し、1配信あたり計7名体制（出演者2名・現場ディレクション＆カメラ4名・配信オペレーション1名）で運営。外部委託なし・準備期間2週間前後で実現。
    *   **BA育成の仕組み化（育成マネージャー専任配置）:** トップ美容部員6名の成功要因を言語化・体系化し、再現性のある育成プランを策定。さらにオンライン活躍を人事評価・インセンティブ制度と連動させることで、属人的スキルへの依存を排除している。

### 頭打ちから読み取れる3つの構造的事実

1.  **視聴者数（リーチ）は「刈り取り上限到達＆微減傾向」にある:**
    2024年度の平均4,034人から2025年度は5,151人へ伸びたが、最大値（max）の上限は2024年度6,934人 → 2025年度6,404人 → 2026.1-3は5,584人と、**むしろ上限が下がっている**。これは集客導線（プッシュ通知・メルマガ等）の刈り取り上限に達していることを意味する。
2.  **エンゲージメント系指標（視聴分数・CTR・コメント率）は2025年度以降、明確に劣化している:**
    平均視聴分数は2024年度8.21分 → 2025年度3.64分（**▼56%**）、商品CTRも5.38% → 1.09%（**▼80%**）、コメント率も2.49% → 0.52%（**▼79%**）と全て大幅低下。
    ただし **中央値ベース** で見ると劣化幅はやや緩やか：平均視聴分数 6.79分 → 3.66分（**▼46%**）、CTR 2.14% → 1.09%（**▼49%**）、コメント率 0.89% → 0.45%（**▼49%**）。つまり 2024年度は特定の高エンゲージ配信（max 27.45分／CTR 24.86%／コメント率 16.57%）が平均を大きく押し上げていた構造であり、通常配信ベースでの劣化は「平均▼56〜80%」ほど劇的ではない。それでも ▼46〜49% の中央値劣化は無視できず、配信本数の増加（2024年度44回 → 2025年度47回）に伴う「番組の量産化・型化の固定」と、新規視聴者の流入比率上昇（後述リピーター比率参照）が要因として考えられる。
3.  **リピーター比率は2025年度に入って上昇基調（3.30% → 4.93%）にあり、ここに伸びしろがある:**
    全体エンゲージメントは下がる一方で、リピーター（active_repeat）比率は明確に上昇トレンド。**「全視聴者を引き上げる」より「リピーターを増やしてエンゲージメントの分母から底上げする」戦略の方が現実的**であることが、データ面から裏付けられている。

### 2026年度の戦略構造（主KPI / 守備KPI の階層）

**「全視聴者（active）のKPIで戦わない」** ことが本提案の核となります。全視聴者の実質上限（直近6ヶ月max）は 6,234人（視聴）／5.80分（分数）／2.48%（CTR）／3.04%（コメント）で、これ以上はマツキヨ本体の集客チャネルでは押し上げ困難です。

一方、**リピーター（active_repeat）は全視聴者比でCTR約3.7倍（3.87% vs 1.04%）・コメント率約5.3倍（3.78% vs 0.71%）**と圧倒的な深度を持ち、母数も伸びています。

よって2026年度は以下の階層構造で KPI 全体を設計します。

```
【主KPI】リピーター系 3指標（3.5 節で目標値を定義）
  A: リピーター比率（active_repeat / active）
  B: リピーターCTR
  C: リピーターコメント率
         ↓ これが上がると
【守備KPI】全視聴者系（3 節の KPI ①〜④）
  ① ライブ視聴者数（リーチ）
  ② 平均視聴分数（リテンション）
  ③ 商品CTR（アクション）
  ④ コメント参加率（エンゲージメント）
  → 主KPI改善の副次効果として自然に改善する水準を「現状維持の防衛ライン」として設計
```

リピーター母数を底上げすれば、彼らが全視聴者の分母に混ざることで、全視聴者ベースの②③④も自然に引き上がる構造です。主KPI（リピーター系）を「攻め」、守備KPI（全視聴者系）を「守り」として役割を明確に分離します。

## 3. 各数値・項目のKPI設計とターゲット値（2026年度・守備KPI）

**本セクションは守備KPI＝全視聴者ベース（KPI ①〜④）** を定義します。主KPI＝リピーター系 3 指標は「3.5. 主KPI：リピーター系」で定義します。

守備KPIは単独で達成を追うものではなく、**主KPI（リピーター系）の改善に伴う副次効果として自然に到達する水準**として設計しています。現状値・過去の天井を踏まえ、STEP1（上期）／STEP2（下期）の目標値は以下のとおり維持しつつ、各目標値の位置付けは「主KPI改善の結果として到達する水準」とします。

### KPI ①（守備KPI）：ライブ視聴者数（リーチ指標）
*   **KPI目標値:** **5,000 人 / 60分あたり** （現状維持・ディフェンス目標）
*   **設計ロジック:** 既に他社ベンチマーク（KANEBOやロート）を遥かに凌ぐ集客力があるため、無理に新規層を追うのではなく、現在のリーチ数を「最低ラインのベース」として安定確保します。主KPI（リピーター系）改善に伴い、同水準を安定維持できることを確認します。

### KPI ②（守備KPI）：平均視聴分数（リテンション指標）
*   **KPI目標値:**
    *   **STEP 1 (上期目標): 6.0 分** （現状の約2倍へ）
    *   **STEP 2 (下期目標): 10.0 分** （ロート製薬・資生堂の下限水準）
    *   *(North Star: KANEBO水準 28.0分)*
*   **設計ロジック:** 全視聴者の実質上限が 5.80分（直近6ヶ月max）であるため、STEP1 の 6.0分は「主KPI＝リピーター比率が6.0%に伸びることで、分母にリピーター層が占める比率が上がり自然に到達する水準」として設計。STEP2 の 10.0分もリピーター比率が 8.0% に伸びた状態での結果指標です。

### KPI ③（守備KPI）：商品クリック率 / CTR（アクション指標）
*   **KPI目標値:**
    *   **STEP 1 (上期目標): 3.0 %**
    *   **STEP 2 (下期目標): 5.5 %** （ロート製薬水準の突破）
    *   *(North Star: KANEBO水準 11.7%)*
*   **設計ロジック:** 全視聴者CTR 1.04% とリピーターCTR 3.87% が混在する現状から、リピーター母数を増やすことで分子・分母構成が変わり自然に上昇する想定。主KPI（リピーターCTR）が 5.5% に伸びれば、全視聴者CTR も 3.0% 台に到達します。

### KPI ④（守備KPI）：コメント参加率（エンゲージメント指標）
*   **KPI目標値:**
    *   **STEP 1 (上期目標): 3.0 %**
    *   **STEP 2 (下期目標): 4.5 %**
    *   *(North Star: KANEBO水準 9.1%)*
*   **設計ロジック:** 視聴分数を上げるための先行指標。主KPI＝リピーターコメント率が伸びれば、全視聴者コメント率も同様に上がる構造。現状（0.71%）から競合同等水準（4.5%）を最終目標とし、将来的にKANEBO（9.1%）水準の双方向コミュニティを築く土台を作ります。

## 3.6. 守備KPI：KPI定義と算出式（読み手向け注釈）

各守備KPIが「何を意味し」「どう算出された数値か」を明示します。社内・先方の双方が同じ基準で進捗を確認できるようにするための定義集です。

### 共通の前提条件

-   **データ元:** Firework管理画面のCSVエクスポート（`active`セクション＝全視聴者ベース／`active_repeat`セクション＝リピーターベース）
-   **集計単位:** 配信1回ごとに数値を取得し、複数配信は単純平均で集計
-   **「60分換算」の定義:** 配信尺がブランドごとに異なるため、`各指標 × 60 / 実配信時間（分）` で正規化。視聴者数・視聴分数のみに適用（CTR・コメント率・エンゲージ率などの比率系は配信時間と独立なので生値）
-   **比較期間:** 競合との比較は全て **2026年1月〜3月の3ヶ月（直近）** に揃えている

### KPI ①（守備KPI）：ライブ視聴者数（リーチ指標）

| 項目 | 内容 |
| :--- | :--- |
| 定義 | 配信中に **1秒以上** 視聴したユニーク視聴者数 |
| 算出式 | `Firework CSV: active セクション「ライブ視聴者数(1秒以上)」 × 60 / 実配信時間（分）` |
| 参照データ元 | 行6（active／ライブ視聴者数(1秒以上)） |
| マツキヨココカラライブ実績 | 2026.1-3 平均 5,084人／60分換算 |
| 注意事項 | 同一視聴者が再入場しても1回としてカウント。最大同時接続数とは別指標 |

### KPI ②（守備KPI）：平均視聴分数（リテンション指標）

| 項目 | 内容 |
| :--- | :--- |
| 定義 | 配信中の **1視聴者あたりの累積視聴時間（分）** の平均値 |
| 算出式 | `Firework CSV: active セクション「平均視聴分数」 × 60 / 実配信時間（分）` |
| 参照データ元 | 行9（active／平均視聴分数） |
| マツキヨココカラライブ実績 | 2026.1-3 平均 3.09分／60分換算 |
| 注意事項 | 「同じ人が複数回入場した場合の合算時間÷ユニーク視聴者数」。Fireworkの公式定義に準拠 |

### KPI ③（守備KPI）：商品クリック率 / CTR（アクション指標）

| 項目 | 内容 |
| :--- | :--- |
| 定義 | **配信中に商品カードをクリックしたユニークユーザー数 ÷ ライブ視聴者数(1秒以上)** |
| 算出式 | `Firework CSV: active セクション「商品クリック率」`（生値・配信時間の影響を受けないため60分換算しない） |
| 参照データ元 | 行12（active／商品クリック率） |
| マツキヨココカラライブ実績 | 2026.1-3 平均 1.04% |
| 注意事項 | 「クリック数」ではなく「クリックしたユニークユーザーの比率」。複数商品をクリックしても1人=1としてカウント |

### KPI ④（守備KPI）：コメント参加率（エンゲージメント指標）

| 項目 | 内容 |
| :--- | :--- |
| 定義 | **配信中にコメント投稿したユニークユーザー数 ÷ ライブ視聴者数(1秒以上)** |
| 算出式 | `Firework CSV: active セクション「コメント参加率」`（生値・60分換算しない） |
| 参照データ元 | 行14（active／コメント参加率） |
| マツキヨココカラライブ実績 | 2026.1-3 平均 0.71% |
| 注意事項 | 1人が複数コメントしても1人としてカウント。チャット数（投稿総数）とは別指標 |

### 補助KPI：エンゲージ率（参考指標）

| 項目 | 内容 |
| :--- | :--- |
| 定義 | Firework公式「エンゲージ率」。**何かしらのアクション（CTR・コメント・いいね等）を行った視聴者の比率** |
| 算出式 | `Firework CSV: active セクション「エンゲージ率」`（Firework内部ロジック） |
| 参照データ元 | 行10（active／エンゲージ率） |
| マツキヨココカラライブ実績 | 2026.1-3 平均 5.11%／全期間平均 18.41%／2025年度平均 7.58% |
| 採用判断 | エンゲージ率は **CTR + コメント率 + いいね率の合計より大きく、Firework内部の独自集計（要確認）** のため、本提案ではKPIとしては採用せず「補助指標」として扱う。代わりに ③CTR・④コメント率 の2つを直接KPIとする |
| 注意事項（要Firework確認） | 「アクティブ率」「エンゲージ率」「アクティブ視聴者数」「エンゲージ視聴者数」の4つの公式定義はFireworkドキュメントに明記がない。担当者経由で正式定義の確認が必要 |

### データ取得・集計時の注意事項

1.  **競合データのスケール混在:** マツキヨCSVは比率を 0-100（%）で記録、競合Excel（資生堂・アットコスメ・ロート製薬）は 0-1（小数）で記録。集計時は自動判定（`max ≤ 1.5 → ×100倍`）でスケールを揃えている
2.  **競合の `active_repeat` データ:** 資生堂・アットコスメ・ロート製薬はxlsx内シート、KANEBOはCSVで社内参考資料に格納されている。当初「非公開」と判断しがちだが必ずここを確認すること
3.  **比較期間の統一:** 競合4社のデータも全て 2026年1月〜3月期に揃えている（時期差による比較ノイズを排除）

## 4. KPI達成に向けたアクションプラン仮説（Next Step）

直近分析によって「驚異的な集客力に反し、配信中の『引き留め』と『誘導』が全く機能していない」ことが明白となりました。これらを解決するための具体アクションです。**4. リピーター育成施策** が主KPI（リピーター系3指標）に直結する打ち手で、1〜3 は守備KPI（全視聴者系）を下支えする配信内施策です。

1.  **「魔の最初の3分」の対策（守備KPI②・平均視聴分数向上へ）**
    *   現状の平均視聴分数が3.09分のため、配信開始から3分以内に「今日紹介する目玉商品」や「最後まで見るともらえる特典等」を最も高い熱量で伝え、即座の離脱を防ぎます。
2.  **アクション誘導の「型」化（守備KPI③・商品クリック率向上へ）**
    *   商品を紹介する際、**「今表示されている〇〇のボタンから詳細が見れます。一旦カートに入れておいてください」**と実演付きでアナウンスを義務化し、CTRの劇的改善を狙います。
3.  **冒頭・中盤での「簡易アンケート」実施（守備KPI④・コメント参加率向上へ）**
    *   「普段〇〇で悩んでる方、1を押して！」「どっちの色が好き？」など、タップだけで答えられる簡易的な投げかけを序盤に仕込み、参加（コメント）への心理的ハードルを下げます。
4.  **リピーター育成施策（主KPI A・B・C の底上げへ・2.5 節結論と接続）**
    *   **定期配信化（曜日・時間帯固定）:** アットコスメの「毎週金曜20時×4年継続」を参考に、視聴ルーティンを形成。ロート製薬の「月1回ロートハートライブ」と同様、頻度よりも「定期性」がリピーター育成に効く可能性が高い。
    *   **リピーター向け限定特典:** 初回視聴者とは別に、2回目以降の視聴者向けクーポン・限定商品・限定企画を用意。再訪インセンティブを具体化する。
    *   **コミュニティ化（継続視聴者同士の交流）:** 「中の人」コメント参加・配信内休憩コーナー・レギュラー視聴者ハンドルネーム認知など、アットコスメ型のコミュニティ運営を導入。リピーター層が「自分の居場所」と感じる設計で、リピーター比率・リピーターコメント率を同時に引き上げる。
