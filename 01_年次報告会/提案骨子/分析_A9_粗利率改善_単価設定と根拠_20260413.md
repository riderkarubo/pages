# A9：粗利率改善｜正しい単価設定と根拠の整理
**作成日：2026年4月13日**
**参照：笹木さんFB（2026-04-13）、マスターデータ、競合調査・KPI設計_提案書.md、A3分析**

---

## 分析サマリー

ライブコマース粗利率69.3%は、他媒体（ショート動画SNSジャック83.3%・Xフォロリポ81.6%）と比較して約12〜14pt低い。
原因を「単価が低い」vs「コストがかかりすぎている」の2軸で分解すると、**主因はコスト構造**にある可能性が高い。
ただし、現状の単価50万円も「メーカーの提供価値に対して割安」の可能性があり、**単価引き上げも並行して検討すべき**。

## 1. 現行の収益・コスト構造

### 現行単価
| 項目 | 金額 |
|------|------|
| 協賛基本単価 | **500,000円/回**（固定） |
| 従量課金（再生回数） | 配信ごとに加算（0〜160,780円/回・2025年度平均約26,000円） |
| **2025年度 平均受注単価** | **約571,000円/回**（26回 / 14,868,840円） |

### 制作工数（1配信あたり）
| 担当 | 工数 |
|------|------|
| MCCM側（メーカーやりとり・台本確認・薬機法チェック） | **10〜20時間** |
| Firework側（台本作成） | **約13時間** |
| **合計** | **約23〜33時間/回** |
| 準備期間 | **1.5〜2ヶ月前から開始** |

> 競合比較：アットコスメ（内製・完全内製・準備期間約2週間）との差が大きい

### 粗利率の推定構造

笹木さん提示の粗利率69.3%から逆算すると：
- 平均受注単価（売上）：約571,000円/回
- **コスト（粗利率30.7%分）：約175,000円/回** の原価がかかっている計算
- 主なコスト候補：Firework制作費・外部ディレクター費・スタジオ・機材・人件費按分

## 2. 原因の二分法：「単価が低い」vs「コストが高い」

### 仮説A：単価が低い

| 根拠（単価が低い可能性） | 詳細 |
|----------------------|------|
| 5,000人超の視聴者リーチ | 競合ライブコマース（資生堂6,958人・アットコスメ体験会390人）と比較しても最高水準。リーチ価値に対して50万円は割安の可能性 |
| アーカイブ再生の長期価値 | 配信後1ヶ月でライブの3.3倍のリーチに拡大。追加コストゼロで露出が増える価値を単価に反映していない |
| リテールメディアとしての購買データ | 会員IDとの突合（CDP）による購買行動データが取得可能。単なる「配信露出」以上の価値がある |
| 競合の協賛単価（要調査） | アットコスメ体験会の協賛費用は不明だが、クローズド・サンプル送付付きで28回/3ヶ月実施。エンゲージ率87%というプレミアム体験への対価として高単価の可能性 |

### 仮説B：コストが高い（より可能性が高い）

| 根拠（コストが高い可能性） | 詳細 |
|------------------------|------|
| 外部委託（Firework台本制作）コスト | 台本作成を外注しているため、競合内製と比べてコストが発生。競合アットコスメは完全内製 |
| 薬機法チェック工数 | 法務確認フローが1.5〜2ヶ月の準備期間を生んでいる。確認工数が人件費として原価に乗っている可能性 |
| MCCM側工数10〜20時間 | メーカーやりとり・台本確認・当日対応等の間接コストが積み上がっている |
| 内製化前のコスト体質 | 2026年10月の内製化完了まで外部費用が継続する構造 |

### 結論（仮説）

```
粗利率低下の主因 = コスト構造（外注＋薬機法工数）
補助因 = 単価が価値対比で割安（引き上げ余地あり）
```

## 3. 単価適正化の根拠（提案骨子）

### リテールメディア・広告としての価値算定

ライブコマースを「デジタル広告」として位置づけると、以下の比較が可能：

| 広告メニュー | 指標 | 試算コスト |
|------------|------|----------|
| **マツここライブ（現行）** | リーチ5,000人/回＋アーカイブ約16,500再生 | **50万円/回** |
| YouTube広告（インストリーム） | CPV約3〜5円/視聴（スキップあり） | 5,000人×5円 ＝ **25,000円**（但し視聴品質が異なる） |
| Instagram投稿（インフルエンサー） | フォロワー10万人規模で20〜50万円/投稿 | 20〜50万円（リーチは多いが視聴継続は短い） |
| **リテールメディア（棚前広告相当）** | 購買直前の接点・購買データ付き | 通常50〜200万円/月 |

> **示唆：リテールメディアとしての価値（購買データ＋リーチ）を組み込むと、現行50万円は低すぎる可能性がある。**

### 単価引き上げシナリオ

| シナリオ | 単価 | 根拠 | 年間売上試算（48回） |
|---------|------|------|-------------------|
| 現状維持 | 50万円 | — | **2,400万円** |
| **シナリオA（推奨）** | **65万円** | リーチ価値＋アーカイブ価値の明示化 | **3,120万円（+720万円）** |

| シナリオC（段階型） | 50万円 → 65万円（上期）→ 80万円（下期） | KPI改善実績と連動して値上げ | — |

## 5. 「メーカーが納得できる単価」の見せ方

笹木さんメモ（いしじまメモ）：「メーカーが納得感のある金額設定」

### 納得感を生む4つの根拠

**① リーチの質（視聴者の購買接点）**
> 「1回の配信で5,000人以上のマツキヨ会員が視聴。購買意向の高いユーザーへの直接訴求。テレビCMと異なり、視聴→クリック→購買のデータが取得可能。」

**② アーカイブの長期露出価値**
> 「配信後1ヶ月で視聴が3.3倍に拡大。一度の制作費で約16,500回以上の再生を獲得。単純CPV換算で約30円/視聴（通常動画広告比で高品質なエンゲージ環境）。」

**③ 購買データとの連動（リテールメディア価値）**
> 「会員IDとの突合により、協賛配信を視聴したユーザーの購買行動を追跡可能。ROI測定ができる数少ない媒体。」

**④ 競合他社との差別化**
> 「エンゲージ改善施策（2026年度）によりCTR・視聴分数が向上すれば、同じ50万円でもより高い購買転換が期待できる。改善ロードマップを提示してメーカーに投資継続の理由を作る。」

## 6. 先方（笹木さん）への回答骨子

**質問：「単価・コストに関する競合の情報をご教示いただけるとありがたい」**

### 回答構成案

1. **まず自己診断を先に示す**
   > 「粗利率差の主因は、現在の外注制作体制（工数23〜33時間/回・準備期間1.5〜2ヶ月）にあると分析しています。2026年10月の内製化完了後、コスト構造は大きく改善される見込みです。」

2. **競合単価は「不明」だが根拠を示す**
   > 「競合他社の協賛単価の正確な情報は入手困難ですが、リテールメディアとしての価値（5,000人リーチ＋購買データ）を基準に、50万円は割安の可能性があります。適正単価の根拠として以下のデータをご提示します。」

3. **単価引き上げ提案（段階型）**
   > 「2026年度は現行50万円を維持しつつ、KPI改善（CTR・視聴分数）が実現した段階で65万円への引き上げをご提案します。内製化完了後（2026年10月〜）はコスト削減と合わせて粗利率80%台を目指します。」
