- 背景帯: x=-1, y=17, w=723, h=52 — `LINEAR_GRADIENT`
- 細い下線: x=0, y=62, w=720, h=1 — `LINEAR_GRADIENT`
- タイトル: x=9, y=20, w=671, h=45 — 透明背景, 白文字
- 該当ページ: P4(似), P9, P41, P44, P46, P47, P50
- 主に「強いメッセージ・カードレイアウト」ページに使用

> 既存ヘルパー `02_helpers.gs` の `TITLE_X=9` は系統Bを想定。系統Aは `TITLE_X=24` で別途扱う必要がある。

### 2-3. フォント色の使い分け

- **黒/濃灰系（`#000000`〜`#333333`〜`#1A1A2E`）**: 通常テキスト
- **`#434343`**: リード文・テーブル本文（少しソフト）
- **`#666666`**: 注釈※（標準ルール）
- **マゼンタ系（`#FA006D` / THEME:ACCENT3）**: 強調インライン
- **白（`#FFFFFF`）**: タイトル黒帯・カードヘッダ帯・KPIテーブルヘッダ

## 4. レイヤー順序のルール

カード系パターンは「**背景 → アクセント → テキスト**」の順で重ねる:

1. ROUND_RECTANGLE 全体背景 (例: 白)
2. RECTANGLE / ROUND_RECTANGLE アクセントバー (左 or 上)
3. 装飾の細帯 (任意)
4. TEXT_BOX 本体テキスト

→ Slides UI で「クリックで選択」できるように、テキストは独立 TEXT_BOX が原則。
ただし「rectTxt」(矩形に直接テキスト埋め込み・1要素化) は単純見出しBOXで可。

## 5. 注釈 (※印) の統一ルール (CLAUDE.mdと一致)

- フォントサイズ **8pt** で統一
- 色 **`#666666`**
- 配置 START / 左揃え
- 「※ 」で始まる文言

例: P15「※ 2025年度（n=47）と直近2026.1-3（n=10）の比較」、P48「※ アットコスメ社価格は…」

## 7. 他案件への転用ルール

このテンプレートをアシックス/ロレアル/サムスン等で再利用する場合：

### 差し替え必要な変数

```javascript
// クライアント色
var BRAND_ACCENT = "#FA006D";     // ← 変更点
var BRAND_GREEN = "#1B998B";       // ← 変更点
var BRAND_NAVY  = "#0F3460";       // ← 変更点

// クライアント名・ロゴ
var CLIENT_NAME = "株式会社MCCマネジメント御中";  // ← 変更点
var LOGO_IMAGE_URL = "...";                       // ← 変更点
```

### 変更不要 (固定値)

- 全座標 (絶対位置)
- フォント (Noto Sans JP / Arial)
- フォントサイズ階層
- カードのROUND_RECTANGLE半径
- ページ番号位置
- 黒帯/グラデ帯の寸法

→ 色変数 5-7個と固有テキストだけで他クライアントに転用可能な設計。
