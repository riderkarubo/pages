## 各スライドの詳細レイアウト案

### S1: エグゼクティブサマリ
- タイトル: 「エグゼクティブサマリ」
- 上段（メインメッセージ）: 「貴社とFireworkが目指すべきビジョンにアラインするメッセージ」（プレースホルダー・Kazuki記入待ち）
- 下段（KPI 2列×2行）:
  - 累計視聴 200万人超（2024+2025）
  - 累計協賛売上 2,935万円
  - 配信回数 91回
  - 平均視聴 5,860人/回（前年比+52%）

### S2: 2025年度総括（章扉）
- タイトル: 「2025年度総括 ― 1年間で積み上げた『資産価値』」
- サブタイトル: 出典：マスターデータ・協賛実績xlsx・SNS数値CSV
- 中央ブロック（2項目を大きく訴求）:
  - 視聴者数 +149%（ライブ+407%・アーカイブ+42%）
  - 協賛実施回数 +44%（18回→26回）

### S3: 視聴者数の急成長
- タイトル: 「視聴者数 +149%（年度比）」
- テーブル5列: 期間 / ライブ累計 / アーカイブ累計 / 合計 / 1回平均（ライブ）
  - 数字横タグで前年比を併記
- フッター注釈: 「※前年比は2024年度実績との比較」

### S4: 協賛実績の推移
- タイトル: 「協賛実績の推移」
- テーブル4列: 期間 / 協賛実施回数 / 売上合計 / 備考
  - 2024年度: 18回 / 14,320,290円 / 単価300万円を2件受注（ブランドジャック型）／基本単価50万円
  - 2025年度: 26回(+44%) / 14,868,840円(+4%) / 基本単価50万円（+従量課金）
- ポイント注釈: 「2024年度は高単価案件2件で売上を積み上げた年。2025年度は件数ベース（26回）で受注を安定化。2026年度は件数KPI確定（48件/年）＋単価見直しでトップライン成長を目指す」

### S5: マイルストーン
- タイトル: 「マイルストーン」
- 縦タイムライン9項目:
  - 2024年4月: 本格稼働スタート
  - 2024年6月5日: 1社協賛 初回配信実施（melt）
  - 2024年8月: オンラインサイトのフローティング表示導入により視聴者数が急増（5月平均約300人→8月以降5,000人超）
  - 2024年8月19日: ブランド横断型 初回配信実施（売れ筋ナプキン徹底比較 / ユニ・チャーム協賛）
  - 2024年11月: 毎月メーカー協賛を安定獲得する体制を確立
  - 2025年10月: 第2期出演者オーディションにより新メンバー3名加入（1期生含め計10名）
  - 2025年10月28日: LIPSコラボ 初回配信実施（KATE）
  - 2026年1月〜5月: 内製化レクチャー開始（全5回）
  - 2026年度 目標: アプリ内ライブLP導線の改修により視聴者数増加予定

### S6: 7つの経営資産
- タイトル: 「7つの経営資産」
- 7カード（横並び・小カード）:
  - 資産01 視聴者資産: 累計約144万人 / アーカイブが3.3倍に拡張
  - 資産02 コンテンツ資産: 91回 / 配信後もアーカイブとして蓄積
  - 資産03 メーカー関係資産: 約19社（のべ）・累計44回 / 月4〜5回広告販売できる体制の構築
  - 資産04 データ資産: CDP上に蓄積中 / 視聴者ID×会員ID突合
  - 資産05 ノウハウ資産: 内製化レクチャー全5回（2026年1月〜5月実施）
  - 資産06 出演者資産: 育成された出演者 計10名（第2期オーディションで3名追加）
  - 資産07 パートナー資産: LIPSとの連携によるリテールメディア強化（メニュー化済み）

### S8: 売上構造の分解（KPIツリー）
- タイトル: 「協賛配信の売上構造」
- ツリー: 売上 = 受注件数（商談×受注率）× 受注単価
- 注釈: Fireworkは「受注率と受注単価」をコンテンツ・メディア強化で押し上げ

### S9: マイルストーン①
- タイトル: 「マイルストーン①：メディアパワー（全視聴者ベース）」
- 4カード（横2×縦2）: 視聴者数（5,084→10,000）/ 視聴分数（3.7→7）/ CTR（1.13→2.2）/ コメント率（0.68→1.0）

### S10: 競合ベンチマーク
- タイトル: 「競合ベンチマーク（コスメカテゴリ・60分換算）」
- テーブル: 5社×4指標
  - A社（リテール）: リピーター50.5% / 視聴24.3分 / CTR29.4% / コメント33.4%
  - C社: 43.1 / 28.2 / 11.7 / 9.1
  - B社: 17.3 / 11.8 / 8.2 / 5.1
  - D社: 16.5 / 9.9 / 5.3 / 4.4
  - マツキヨ: **5.0%** / 3.1分 / 1.0% / 0.7%
- 強調: マツキヨ行を異なる色で目立たせる

### S11: メディアパワーの因数分解
- タイトル: 「メディアパワーの因数分解 ― リピーターが最大レバー」
- 上段（中央）: 数式バー
  - **メディアパワー = 集める力 × 引き留める力 × 再訪する力**
  - 注釈: （= 視聴者数 × エンゲージメント深度 × リピーター比率）
- 下段（3カード横並び）:
  - ① 集める力 — 業界上位・頭打ち（5,000人超を安定維持）
  - ② 引き留める力 — 競合と大差・伸び代大（視聴分数・CTR・コメント率すべてで競合トップの2〜11%水準）
  - ③ 再訪する力 — ②を底上げするレバー（リピーターはCTR/コメント率が非リピーターの数倍）
- フッター注釈: 「集める力は業界上位。次は引き留める力と再訪する力を作る年」

### S12: マイルストーン②
- タイトル: 「マイルストーン②：リピーター数値（追加提案）」
- 3カード: リピーター比率（4.88→8.0）/ リピーターCTR（3.87→8.0）/ リピーターコメント率（3.78→7.5）

### S14: 柱① 現状課題
- タイトル: 「柱① 現状課題」
- 2カード:
  - ① 競合と比べて価格が低い（A社15分=45万、マツキヨ60分+データ連動=50万）
  - ② 貴社の他メニューと比較して粗利率が低い（69.3%、ショート動画83.3%・Xフォロリポ81.6%）

### S15: 柱① 提案①：単価是正
- タイトル: 「提案①：フルサポートプラン単価是正（A社比較）」
- テーブル: A社スタンダード45万（15分・3商品）/ A社ポップアップ70万 / マツキヨ現行50万（60分・データ連動・5,000人）
- 結論: 同水準の価格で60分配信＋データ連動を提供しており、**市場価格との乖離がある**
- ※グレーアウト済みの「→65万円」「粗利率インパクト」「広告換算」「提案②③」「メニュー3層」は本資料に非掲載

### S16: 柱① 体制改善 現状課題
