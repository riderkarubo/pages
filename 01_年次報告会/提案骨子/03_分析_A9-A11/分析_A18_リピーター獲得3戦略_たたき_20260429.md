## 1. 全体の前提整理

| 項目 | 現状（マツキヨ） | 競合トップ | ギャップ |
|---|---|---|---|
| リピーター比率 | 5.0% | A社50.5% / C社43.1% | 約8〜10倍 |
| 強み | 集める力（5,000人/回） | — | 既に業界上位 |
| 弱み | 「戻り客を作る」思想が配信設計に未実装 | LINE通知/個人ファン/ブランドコミュニティ | 思想・導線・体験の3つすべて |
| 武器 | CDP（視聴者ID×会員ID突合）／LIPS連携／公式アプリ | — | 競合は持っていない構造的優位 |

世界に存在するリピーター獲得の3つの型：

1. **「人で再訪させる」型**（資生堂モデル）：個人インフルエンサー化したBAのファンを呼ぶ
2. **「ブランドで再訪させる」型**（KANEBOモデル）：ブランド愛好家コミュニティが既に存在
3. **「番組で再訪させる」型**（@cosmeモデル）：定期番組としての構成と習慣化

→ マツキヨは**3つ目の型をベースに、1つ目と2つ目を補強する複合戦略**が最も現実的。

## 2. 戦略①：「推しBA」設計 ― 人で連れ戻す（資生堂×Twitch型）

### 狙い
「商品を見に行く」から「あの人を見に行く」へ動機を転換。リピート理由を出演者のキャラクターに固定する。

### 根拠（既存資料）
- 資生堂：個人インフルエンサーBA 14名で累計99万人フォロワー → リピーター比率17.3%（マツキヨの3.5倍）
- A1分析：マツキヨのリピーターはCTRが非リピーターの**3.7倍**、コメント率は**5.3倍**。1人増やすと指標が一気に動く
- 既に提案HTMLに「柱② 社員インフルエンサー強化」として土台あり → これを"具体化"する位置づけ

### 外部ノウハウ
- **Twitch 2026モデル**：「視聴回数より、チャット活動・watch time・常連認知が新規流入を呼ぶ」。雑談中に "○○さん今日も来てくれてありがとう" を入れるだけで再訪率が劇的に上がる
- **QVCの "Trusted Advisor"**：常連顧客が「ホストのことを友達のように知っている」状態が30年続く。ホストは商品ではなく"関係"を売っている
- **Sephora SEPHORiA**：Beauty Insider限定の virtual event を年1回開催し、ブランドファウンダーとの1対1セッションを"報酬"として提供

### マツキヨへの具体施策（4本）
1. **「レギュラー化」**：10名の出演者から **常設メイン3〜4名 + ローテ6〜7名** の構成にし、「火曜=Aさん、第2週=Bさん」と曜日固定。視聴者が「見たい人の日」を覚えられる構造
2. **個人SNS解禁＋"中の人"運用ガイドライン整備**：会社アカウントではなく個人名アカウントで配信前後にティザー・裏話を投稿。薬機法CKは事後アーカイブ確認の運用に切り替え（資生堂PoCと同じ）
3. **配信内「常連呼びかけ」プロトコル**：冒頭2分で「先週来てくれた○○さんありがとう」、終盤「次回火曜は△△さんが××を紹介します」を必須シーケンス化。台本テンプレに組み込む
4. **出演者プロフィールページの新設**：マツキヨアプリ／ECサイトに各出演者の「推し商品履歴・配信アーカイブ・次回出演予定」を集約。"推し活"の導線を作る

### サマリー
| 項目 | 内容 |
|---|---|
| コスト感 | 低〜中（既存リソースの運用変更が中心、SNSマネジメント工数のみ追加） |
| 効果発現 | 3〜6ヶ月 |
| リピーター比率インパクト | +1.5〜2pt（資生堂初年度水準を参考） |
| マイルストーン①への副次効果 | コメント率↑ |
| 既提案との接続 | 柱② 社員インフルエンサー強化の具体化 |

## 3. 戦略②：「マツキヨ×Firework のロイヤリティ・ループ」 ― CDPで連れ戻す（マツキヨ独自・他社不可能）

### 狙い
競合が持たない **CDP × 公式アプリ × 全国会員基盤** を活用し、「視聴×購買×ポイント」を1本のループにつなげる。これは資生堂・KANEBO・@cosmeも持っていないマツキヨ独自の構造的優位。

### 根拠（既存資料）
- 資産04データ資産：**CDP上に視聴者ID×会員ID突合済み**（提案HTMLで明記）
- KANEBO：LINEミニアプリ「スマイルコネクト」で開封率70%・関連リンクCTR10〜15% → 通知設計だけでこの効果。マツキヨは公式LINEアカウント保有
- 業界ベンチマーク：通常ECのリピート率20〜30%、ライブコマースは60〜70%（commercepick）。マツキヨ5%は明らかにロイヤリティ設計の不在

### 外部ノウハウ
- **Sephora Beauty Insider × SEPHORiA**：会員ランクごとに virtual event 招待、参加で loyalty points 付与。ライブを「ポイントが貯まる/使える場」に位置付け
- **QVC**：既存顧客の年間平均購入32点・$1,647。"買い続ける動機"の中心はホストとの関係 + 限定商品の組み合わせ
- **Lifestyle (MoEngage事例)**：ダイナミック・エンゲージメントで30%リテンション改善。トリガーは「行動ベースの個別通知」

### マツキヨへの具体施策（5本）
1. **「マツキヨアプリ会員限定」ライブ視聴特典**：会員ログイン状態で視聴 → ポイント2倍 / 限定クーポン / アーリーアクセス。"無会員より得"な状態を作る
2. **次回配信のパーソナライズ通知**：CDPで「過去視聴ジャンル」「直近購買履歴」と次回配信内容をマッチング → 公式LINE/アプリPushで個別配信。一斉送信ではなく1to1
3. **「視聴ストリーク」ゲーミフィケーション**：3週連続視聴で限定バッジ、5回でサンプル配布、10回で出演者からの直筆メッセージ。Sephora の POAP/NFT 的体験のローカル版
4. **配信終了20秒で発火する「次回予約」ボタン**：ワンタップでカレンダー登録 + LINE通知ON。視聴熱量がピークの瞬間に予約UIを差し込む（Twitch のフォロー導線と同じ思想）
5. **「ライブ限定SKU/バンドル」の常設化**：配信中に「ここでしか買えない3点セット」を毎回必ず1点用意。資生堂のライブ限定500円クーポンの上位互換

### サマリー
| 項目 | 内容 |
|---|---|
| コスト感 | 中〜高（CDP連携設計・アプリ改修・LINE個別配信の運用構築） |
| 効果発現 | 3〜9ヶ月（段階導入） |
| リピーター比率インパクト | +1〜2pt（KANEBO LINEミニアプリ事例の保守的見積） |
| マイルストーン①への副次効果 | CTR↑ |
| 既提案との接続 | 新規軸（CDP活用）。ただしマツキヨ独自性として最も強い |

## 4. 戦略③：「番組として観に来てもらう」 ― フォーマットで連れ戻す（@cosme×アットコスメ型）

### 狙い
単発の商品紹介ライブの集合体ではなく、**1本の連続テレビ番組**として設計し直す。視聴者の頭の中に「今週もマツコココライブ観なきゃ」という習慣を作る。

### 根拠（既存資料）
- @cosme：毎週金曜20時を4年継続 → リピーター比率50.5%（業界1位）。「セルフチェック機能」「原因解説」の構成設計が異次元のCTR49.4%を生んでいる
- A5フォロリポ分析：チャット参加が▲60〜67%増加 → "参加型の場"設計でリピーターは確実に育つ
- マツキヨは火曜17:55の定期配信は実施中だが「番組として識別される構成」が未確立

### 外部ノウハウ
- **MMD研究所調査**：ライブコマース視聴経験者の **32.3%が"ほぼ毎月視聴"** している（=習慣化が成立する市場）
- **Twitch 2026**：「常連を認知する＋繰り返しテーマ／インジョーク／お決まりコーナー」が retention rate を最も上げる
- **QVC Today's Special Value**：毎日1点だけ24時間特価という"番組性"が38年続くフォーマットの核
- **Deloitte 2026 Digital Media Trends**：「ファンダムを高価値セグメントとして年間通じてケアする」モデルが LTV 最大化に最も寄与

### マツキヨへの具体施策（5本）
1. **「コーナー制」導入**：60分を5〜6コーナーに分解 ― ①今週の新発売 / ②視聴者悩み相談 / ③推しBAの私物紹介 / ④メーカー担当者ゲスト / ⑤ライブ限定SKU / ⑥次回予告。各コーナーに固定タイトルとジングル
2. **「視聴者の悩み診断」コーナー**：@cosmeのセルフチェックを参考に、配信中にコメントで悩みを募集→出演者が即興で商品提案。"自分ごと化"の最強装置（CTR改善にも直結）
3. **メーカー担当者の「中の人」化**：協賛メーカーの開発担当者・PR担当者をコメント欄に固定参加させる。視聴者からの質問にリアルタイム返答 → @cosmeが実証済みの方法。協賛メーカー側にとっても "出演機会" として売れる
4. **「火曜ナイト常連クラブ」の世界観構築**：番組ロゴ・ハッシュタグ・専用ジングル・常連あだ名（"火曜部"等）を整備。「マツキヨの新コーナー」ではなく「独立した番組ブランド」として打ち出す
5. **アーカイブの能動的活用**：終了直後に X / アプリ内に「3分ハイライト」を即日アップ。次回までの"間"を持たせる。次回告知ティザーも同じ枠で

| 項目 | 内容 |
|---|---|
| コスト感 | 低〜中（演出・台本構成のリデザイン、ジングル/ロゴ等の素材制作） |
| 効果発現 | 1〜3ヶ月（最速） |
| リピーター比率インパクト | +1〜1.5pt（最も低リスクで効果が早く出る） |
| マイルストーン①への副次効果 | コメント率↑・視聴分数↑ |
| 既提案との接続 | 番組演出強化として新提案 |

## 5. 3戦略の比較とおすすめの組み合わせ

| | ①推しBA | ②ロイヤリティループ | ③番組化 |
|---|:---:|:---:|:---:|
| 投資額 | 低〜中 | 中〜高 | 低〜中 |
| 効果発現 | 3〜6ヶ月 | 3〜9ヶ月 | **1〜3ヶ月（最速）** |
| リピーター比率インパクト | +1.5〜2pt | +1〜2pt | +1〜1.5pt |
| 競合との差別化 | 資生堂モデルの追随 | **マツキヨ独自・他社不可能** | アットコスメ追随＋ローカライズ |
| マイルストーン①への副次効果 | コメント率↑ | CTR↑ | コメント率↑・視聴分数↑ |
| 既提案との接続 | 柱②社員インフルエンサーの具体化 | 新規軸（CDP活用） | 番組演出強化として新提案 |

### 推奨フレーミング

- **上半期**：③番組化（最速効果）＋ ①推しBAの導入（既提案の具体化）
- **下半期**：②ロイヤリティループ（最大インパクト・マツキヨ独自性）
- 3つを並列でやらず、③→①→②の順で立ち上げる方が現実的

→ 合計 **+3.5〜5.5pt** のリピーター比率インパクトが見込め、目標 5.0% → 8.0% は**達成可能性が見える数字**になる
