## エグゼクティブサマリ（冒頭 / 1スライド）

> **「マツここライブは、配信する媒体から資産を積み上げるプラットフォームへと進化した。」**

2年間で91回の配信・累計約144万視聴を達成。しかしこの数字の本質は「配信回数」ではなく、**1回の配信が1ヶ月後に2〜3倍のリーチを生み続けるストック型資産**にある。

```
SNS告知（認知）─→ ライブ配信（体験）─→ アーカイブ（蓄積）
                                              ↓
                         切り抜き動画・SNS再配信（次の施策）
```

| 三位一体の数値 | 実績 |
|-------------|------|
| 1配信あたりライブ視聴者数 | 約5,860人（2025年度平均） |
| 1配信あたりアーカイブ視聴 | ライブの**2倍以上**が配信後に視聴 |
| 合算タッチポイント/回 | 約**12,500回** |
| 協賛配信44回の累計リーチ | **約2,919万円の収益基盤**を構築 |

**2026年度の問い：この循環の「回転速度」をどう上げるか。**

## 柱① 2025年度総括：1年間で積み上げた「資産価値」の報告（5分）

### 報告する主要数値

| 指標 | 数値 |
|------|------|
| ライブ累計視聴者数（2025年度） | 275,403人（前年比 +52%） |
| アーカイブ含む累計視聴（2年間） | 約144万人（1,441,474人） |
| アーカイブ倍率 | **ライブ視聴の2倍以上がアーカイブで視聴**（コンテンツが働き続ける） |
| 協賛メーカー数（のべ2年間） | 約19社 |
| 協賛配信回数（のべ2年間） | 44回（2024年度18回 / 2025年度26回） |
| 協賛売上累計（2年間） | 約2,919万円（2024年度1,432万円 / 2025年度1,487万円） |

### 報告項目

- **配信実績・視聴者推移**：ライブ・アーカイブ視聴数の伸長、コンテンツ資産蓄積状況
- **「蓄積する資産」の証明**：91本の動画ライブラリ × 平均アーカイブ12,500回 ＝ 約114万回の総再生資産
- **メーカー実績**：参画メーカー数の増減・継続率・新規獲得状況、売上インパクト事例
- **7つの経営資産の整理**
  1. 視聴者（累計144万人 ＋ アーカイブが3.3倍に拡張）
  2. コンテンツ（91本・配信後も伸長。melt回は1ヶ月後に2.2倍）
  3. メーカー関係
  4. データ
  5. ノウハウ
  6. 出演者
  7. パートナー

## 柱③ 戦略方針：2026年度KPI達成のための施策（15分のうち後半）

**大テーマ：リテール収益を向上させるためには？**

### 協賛配信の品質・収益改善

| カテゴリ | 施策 | データ根拠 |
|---------|------|----------|
| 協賛配信の品質向上 | 協賛配信の回数増・単価向上 | — |
| 制作支援 | Fireworkによる台本制作支援の活用 | — |
| コンテンツ展開 | **切り抜き動画オプションの作成（過去提案済み、再提案）** | **アーカイブが1ヶ月後に平均2倍超再生（melt回：2日後8,920回→1ヶ月後19,673回）。切り抜きでSNS再配信すれば循環が太くなる** |
| ベストプラクティス共有 | 社内横展開・新メニュー開発 | — |
| 店舗活用 | 詳細はディスカッションへ | — |

### 配信品質向上：具体アクション

| カテゴリ | 施策 | データ根拠 |
|---------|------|----------|
| 早期離脱対策 | **「魔の最初の3分」対策**（冒頭で目玉商品・特典を最高熱量で提示） | 現状平均視聴分数3.1分。商品紹介すら届いていない可能性 |
| CTR向上 | **CTR誘導の「型」化**（「今表示されている〇〇ボタンから…」の実演アナウンス義務化） | 現状CTR1.0%（ロート5.3%、KANEBO 11.7%） |
| エンゲージ向上 | **冒頭・中盤の「簡易アンケート」**（「1を押して！」等でコメント参加ハードルを下げる） | 現状コメント率0.8%。アットコスメは19%超 |
| 配信設計 | **目的別の配信尺使い分け**（〜60分=CTR重視 / 75分〜=課金重視4.2倍 / 65〜73分=バランス型） | 41回実績分析より |

### 月5回以上の受注を実現するために

| カテゴリ | 施策 |
|---------|------|
| 業務効率化 | 現場の工数洗い出し → コンサル受注を狙う |
| AI活用 | 薬機法チェッカーの作成など |

## ディスカッション議題（25分）※先方パート

1. **&ST様との施策に対する明確なプラスの提示**：Firework独自プラットフォーム・グローバル知見の活用で何がプラスオンされるか
2. **収益化の具体スキーム**：動画素材のメーカー販売等、米国成功事例を踏まえた体制・収益モデル
3. **システム導入の親和性と必要性**：階層化承認システム等と既存ツールの機能重複・適合性の検討
4. **（追加）アーカイブ資産の活用・収益化**：「1配信の投資が1ヶ月後に4倍超のリーチを生む」という事実を協賛メーカーへどう訴求するか。切り抜き動画メニュー化の収益シミュレーション
5. **（追加）SNS告知計測体制の再整備**：2025年度のX告知インプレッション・エンゲージメント率が全欠損している現状を踏まえ、2026年度は計測体制を復旧し、告知→ライブ転換率をKPI化する
