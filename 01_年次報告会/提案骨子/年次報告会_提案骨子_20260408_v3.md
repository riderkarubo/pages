## 柱① 2025年度総括：1年間で積み上げた「資産価値」の報告（5分）

### 報告する主要数値

| 指標 | 数値 |
|------|------|
| ライブ累計視聴者数 | 275,403人（前年比 +62%） |
| アーカイブ含む累計視聴 | 約144万人 |
| 協賛メーカー累計 | 約19社 |

### 報告項目

- **配信実績・視聴者推移**：ライブ・アーカイブ視聴数の伸長、コンテンツ資産蓄積状況
- **メーカー実績**：参画メーカー数の増減・継続率・新規獲得状況、売上インパクト事例
- **7つの経営資産の整理**
  1. 視聴者
  2. コンテンツ
  3. メーカー関係
  4. データ
  5. ノウハウ
  6. 出演者
  7. パートナー

## 柱③ 戦略方針：2026年度KPI達成のための施策（15分のうち後半）

**大テーマ：リテール収益を向上させるためには？**

### 協賛配信の品質・収益改善

| カテゴリ | 施策 |
|---------|------|
| 協賛配信の品質向上 | 協賛配信の回数増・単価向上 |
| 制作支援 | Fireworkによる台本制作支援の活用 |
| コンテンツ展開 | 切り抜き動画オプションの作成（過去提案済み、再提案） |
| ベストプラクティス共有 | 社内横展開・新メニュー開発 |
| 店舗活用 | 詳細はディスカッションへ |

### 月5回以上の受注を実現するために

| カテゴリ | 施策 |
|---------|------|
| 業務効率化 | 現場の工数洗い出し → コンサル受注を狙う |
| AI活用 | 薬機法チェッカーの作成など |
